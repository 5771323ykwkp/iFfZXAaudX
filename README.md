# 家政服务管理平台系统介绍 java83

## 项目概述

家政服务管理平台是一个集成了管理员、用户和服务人员三种角色的在线服务平台。该平台基于前后端分离的设计模式，前端采用Vue、Ajax和Json技术，后端采用SpringBoot和MyBatisPlus框架。

## 功能模块

### 管理员功能模块

管理员具备最高权限，可以进行以下操作：

- 首页管理：更新首页内容。
- 个人中心：管理个人信息。
- 用户管理：维护用户信息。
- 服务人员管理：管理服务人员资料和工作状态。
- 服务信息管理：发布与更新服务信息。
- 服务类型管理：分类管理服务。
- 服务预约管理：处理预约请求。
- 服务取消管理：处理取消请求。
- 服务分配管理：分配服务任务。
- 服务进度管理：跟踪服务任务进度。
- 评价信息管理：管理用户评价。
- 留言反馈管理：处理用户留言和反馈。
- 系统管理：进行系统设置。

### 用户功能模块

用户可以执行以下操作：

- 首页访问：浏览首页内容。
- 服务信息浏览：查看服务详情。
- 服务预约：预约所需服务。
- 服务取消：取消预约服务。
- 服务进度查询：查看服务进度。
- 评价信息提交：对服务评价。
- 留言反馈：提交留言和反馈。
- 个人中心：管理个人信息。
- 我的收藏管理：管理收藏服务信息。

### 服务人员功能模块

服务人员可以进行以下操作：

- 个人中心：管理个人信息和服务任务。
- 服务分配管理：查看指派的服务任务。
- 服务进度管理：更新服务任务进度。
- 评价信息管理：查看用户评价。

## 系统角色

- 管理员：负责整个平台的管理和运营。
- 用户：使用平台服务的普通用户。
- 服务人员：提供家政服务的服务人员。

## 运行环境

- 前端：基于Vue的Web应用程序。
- 后端：基于SpringBoot的Java应用程序。
- 数据库：兼容MyBatisPlus的数据库。
